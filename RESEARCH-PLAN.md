# Research plan — tackling the framework's major problems

*A prioritized, honest plan. The organizing principle: the framework lives or dies on **one** question — can it produce a distinctive, testable prediction? Everything else is a cheap gate, an achievable side-win, a structural limit to accept, or conceptual maturation. Each tier has a **decision point** so we learn cheaply whether to continue.*

---

## The strategic logic (read this first)

The 12 problems are not equal. Most of the framework's fate rests on a single **critical path**:

> **mechanism** (force thermal time from the stiffness, Problem 3) → **prediction** (derive the interference size $\kappa$ from the stiffness, Problem 1).

If that path succeeds and $\kappa$ is measured, the framework becomes physics. If it fails, the framework remains an elegant synthesis — regardless of how many other conjectures are proven. So the plan is built around that path, with the rest as **(0)** cheap prerequisites, **(1)** an achievable morale/credibility anchor, **(3)** structural limits to *state* rather than *fix*, and **(4)** conceptual work that can wait.

## Tier 2 — The critical path (1–3 yrs; needs collaborators) — *addresses Problems 1, 3, 5*

**2A. Force thermal time in the GV setting (the mechanism).** Ingredients already in hand: the GV flow is *geometrically canonical* (Connes: flow of weights = GV), removing the "which flow?" objection; and the entanglement-equilibrium / first-law mechanism (Jacobson; Faulkner; Casini) already derives Einstein dynamics from $\delta S = \delta\langle H_{\text{mod}}\rangle$. 
**Target:** prove that the canonical GV/modular flow *is* the physical evolution, via a stiffness-equilibrium (KMS) variational principle — i.e. "minimizing geometric tension = extremizing GV = the equilibrium condition." 
**Needs:** a modular-theory + constructive-QFT collaborator. 
→ **Decision point:** if thermal time cannot be forced here after honest effort, Plank IV stays an interpretation and the critical path is broken.

**2B. Derive $\kappa$ (the fingerprint).** Given 2A, compute the *magnitude* of the nonlinear / Born-rule deviation — Sorkin's triple-path interference $\kappa = I_3/\sum|I_2|$ — from the GV stiffness / topology-change scale (using the gravitize-the-quantum / metastring formalism of Minic et al.). 
**Target:** a specific, currently-testable $\kappa$, distinctive and near-parameter-free, in *quantum foundations* (sidestepping every cosmological degeneracy). 
**Needs:** quantum-foundations + metastring expertise; contact Minic's group. 
→ **Decision point (the big one):** if $\kappa$ is unmeasurably small, or matches the standard-QM null, the framework has **no distinctive prediction** — it is an interpretation. If $\kappa$ is in the measured/soon-measurable window (photon multi-slit, JUNO neutrinos, atom interferometry), the framework becomes **testable physics**.

**2C. (contingent on 2A) Skeletal → foundational.** If 2A works, test whether the same equilibrium principle promotes the protected "skeleton" to the full dynamics (Problem 5) — i.e. recovers more than topology. This is the deepest payoff and is entirely downstream of 2A.

## Tier 3 — Structural limits: *state honestly, don't pretend to fix* — *addresses Problems 2, 4*

- **The global-hyperbolicity divide (Problem 2)** is a theorem (Bernal–Sánchez), not a bug to fix: the rigorous corner and the exotic corner are disjoint. **Plan: state it plainly** as a known limit. *Only if Tier 2 succeeds* is it worth attacking the hard frontier (superselection theory on non-globally-hyperbolic / singular spacetimes — mostly unbuilt).
- **The Hodge-weight obstruction (Problem 4)** is also a theorem; the *escape is the exotic-smoothness channel*, which is exactly the Tier 2 path. Don't try to remove the obstruction; pursue the escape.
- **SUSY-inescapability of the rigorous sector:** accept and state — "geometry-first *on the protected stratum*" is the honest surviving claim, weaker than the original bet.

## The honest bottom line

This plan is really **one bet with cheap insurance around it.** The bet: can Tier 2 (mechanism → $\kappa$) produce a *measured, distinctive number*? The insurance: Tier 0 (learn cheaply if it's novel at all), Tier 1 (bank one real result regardless), and Tier 3 (state the structural limits so we're never accused of hiding them). Sequenced this way, we discover whether to continue at each gate — rather than sinking years before finding out. Odds remain long; but the plan is honest, and its first two tiers are cheap, decisive, and worth doing whatever the grand bet's fate.

**Recommended immediate next action:** Tier 0.1 (scholarship gate) + start Tier 1.1 (the label theorem) in parallel — both are in-room-startable, cheap/decisive, and gate everything else.
