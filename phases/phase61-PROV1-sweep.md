## 3. The broken [proved] tag

**CORE §2.6 / abstract: "Exotic smoothness is physical — it sources gravity [proved]" — the tag is not defensible.** All three citations are real (Brans's *conjecture*, CQG 11 (1994); Asselmeyer's 1997 CQG paper — a physics-style formal derivation, no theorem-proof structure; Sładkowski 2001 — an isometry-obstruction argument he himself frames as a *weakened* Brans conjecture; no Einstein-equation solution with exotic-smoothness stress-energy exists anywhere). The "confirmed" gloss circulating is self-citation within the Brans–Asselmeyer–Sładkowski circle. **Corrected tag: [conjectured (Brans); supported by formal arguments (Asselmeyer 1997; Sładkowski 2001); not proved].** Impact check: nothing in the phases 38–60 spine uses §2.6 — the tag was rhetorical load, now honestly lightened.

## 4. The false mathematics in CORE §4.4 ["why 4D is special"]

Both pillars of the section's argument are **wrong as stated**: (a) *"Cheeger finiteness holds cleanly for n ≠ 3,4"* — false; Cheeger's theorem holds in **all** dimensions (the dim-4 proof-technique gap was closed by Peters 1984). (b) *"4-manifolds admit uncountably many exotic structures"* — false for compact ones (at most countably many; countably infinite occur — Dolgachev/Donaldson); **uncountably many only for open 4-manifolds** (exotic ℝ⁴ — Taubes). **The repair:** the section's conclusion (dimension 4 is uniquely wild) is true and famous, but via the correct statements — ℝ⁴ is the only ℝⁿ with exotic smooth structures, and it has uncountably many — which is, conveniently, the fact the framework's original exotic-ℝ⁴ channel actually used. §4.4 to be rewritten on the correct route in CORE v5; the Cheeger-exception framing is dead.
